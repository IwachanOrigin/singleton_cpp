# Singleton cpp demo code

## Overview
Simple implementation example of singleton pattern using cpp

## Build
```
$ docker-compose build
```
```
$ mkdir /singleton_dev/src/build
$ cd /singleton_dev/src/build
$ cmake ..
```
```
$cmake --build .
```

## Usage
```
$ ./main
```
